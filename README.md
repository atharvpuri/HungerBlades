# Hunger Blades: Exploration Prototype

## Project Overview
This repository hosts the initial prototype for a AAA-level exploration game developed in Unreal Engine 5. The project focuses on high-fidelity environmental design and character traversal mechanics. The player controls a knight character navigating a highly detailed, open-world terrain.

## Development Context
* **Engine:** Unreal Engine 5
* **Genre:** Open-World RPG / Exploration
* **Developer:** Atharv Puri

## Features
* **AAA Environment:** Features complex terrain generation and realistic lighting systems typical of high-end graphical productions.
* **Character Controller:** Functional knight character with exploration capabilities.
* **Systems:** Basic navigation and collision systems implemented within the Unreal framework.

## Download and Installation
Due to the high fidelity of the assets, the project file size exceeds standard repository limits. The full project source, including Content, Config, and the .uproject file

### How to Run
1.  Download the ZIP file from the link above.
2.  Extract the contents to a local directory.
3.  Launch Unreal Engine 5 (ensure you have the latest version installed).
4.  Select "Browse" and locate the `.uproject` file within the extracted folder.
5.  Wait for shaders to compile upon the first launch.
